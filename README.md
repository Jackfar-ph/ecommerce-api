A simple and powerful e-commerce dashboard built with **Node.js, Express, and MongoDB**.

## Features
* **CRUD Operations:** Create, Read, and Delete products.
* **Live Search:** Filter products instantly by name.
* **Inventory Stats:** Real-time calculation of total stock and value.
* **Clean UI:** Responsive two-column dashboard.

## Setup
1. Run `npm install`
2. Make sure MongoDB is running locally.
3. Run `node index.js`
4. Visit `http://localhost:3000`
